# 📊 Amazon Best Seller Analysis Dashboard

## 📌 Project Overview

This Power BI project analyzes Amazon Best Seller data to uncover valuable insights into product performance, customer ratings, pricing, reviews, and popular product categories. The interactive dashboard helps users explore trends and make data-driven decisions using visual analytics.

---

## 🎯 Objectives

- Analyze Amazon Best Selling products.
- Identify the most popular product categories.
- Compare product ratings and review counts.
- Examine pricing trends across categories.
- Create an interactive dashboard for better decision-making.
